A responsive admin dashboard project built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

- Fully responsive layout
- Modern dashboard UI
- Sidebar navigation

## 🛠️ Built With

- CSS3 (Flexbox & CSS Grid)
